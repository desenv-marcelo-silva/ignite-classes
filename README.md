## Run Locally

Clone the project

```bash
  git clone https://github.com/desenv-marcelo-silva/ignite-classes.git
```

Go to the project directory

```bash
  cd ignite-classes
```

Install dependencies

```bash
  yarn
```

Start the server

```bash
  yarn dev
```
